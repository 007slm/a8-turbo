# 实施计划: 原生悬浮窗迁移

## 概述

本实施计划将 A8Wisper 悬浮窗系统从 Tauri webview 渲染迁移到纯原生 Python 渲染。核心原生悬浮窗组件已完成实现，现在需要完成主应用集成和 Tauri 依赖清理工作。
